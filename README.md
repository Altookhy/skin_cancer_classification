# Skin Lesion Classification (HAM10000)

Quick steps to install dependencies and run the notebook.

## Install
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install --upgrade pip
pip install -r requirements.txt
```

## Run
```powershell
code dataset.ipynb   # or open in Jupyter/VS Code and run all cells
```
